Quidditch Evolution

Quidditch Evolution est un jeu multijoueur dans lequel deux équipes s'affrontent. Le principe est semblable à celui du football : il faut marquer plus de buts que l'équipe adverse pour gagner. Le jeu est donc simple dans le sens où il reprend un principe de jeu universel. Cela permet aux joueurs de facilement comprendre le jeu. Les mécanismes du jeu que nous avons apportés, quant à eux, créent une nouvelle expérience pour les joueurs, en ajoutant un coté stratégique très important, aussi bien individuellement qu'en équipe.
